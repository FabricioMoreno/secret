Solution

The solution is pretty simple, consists of finding out the letter of the alphabet having its position encrypted.

Each letter is encrypted like this:

( letter | symbol | number )

The symbol can be plus or minus and depending on whether you go back or advance in the alphabet.

Theory

(current position in the alphabet - number of letters to go back)

(current position in the alphabet + number of letters to advance)

Example

  • ( o + 1 ) : The letter is p
  • ( j + 2 ) : The letter is l
  • ( c - 2 ) : The letter is a
  • ( z - 1 ) : The letter is y
